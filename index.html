  • 추천 대상: 스마트폰 사용량이 많아 5G 고속 인터넷이 필요한 직장인 또는 학생.

  • 추천 대상: 주로 와이파이 환경에서 생활하고 통화 위주로 사용하는 주부 또는 어르신.

데이터 쉐어링/테더링 (가족 결합)

  • 특징: 메인 회선에서 제공되는 데이터를 가족끼리 나눠 쓰는 상품.
  • 장점: 5G 데이터를 넉넉하게 쓰고 싶지만 각각 무제한 요금제를 쓰기 부담될 때 효율적입니다.
  • 추천 대상: 2~3인 가구로, 1명은 무제한을 쓰고 나머지는 데이터를 나눠 쓰는 경우.
